# **sapHowmuch** for SAP Business One

This repository is for developing SAP Business One addon.

Test project is tested on SAP Business One 9.2 PL07.

## Prerequisites

* .NET Framework (4.6.1)
* SAP Business One SDK
* Reactive Extensions
* Winforms

## Further skills

* MassTransit
* RabbitMQ
* ASP.NET WebAPI 2
* VSIX
